# 🛰️ Bridge

> ⚡ 一款基于 **Electron + Vite + Vue3** 构建的跨平台局域网协作工具，为你提供快速、私密且智能的本地网络通信体验。

**Bridge** 让你的局域网变成一个小型智能生态系统 —— 设备发现、文件同步、共享下载，一键即可完成。

[English](./README.md) | 简体中文

## ✨ 功能特性

- 📁 **文件同步与比对**
   对比本地与远程（FTP）目录树的差异，并智能同步文件。
- 🌐 **局域网文件共享**
   基于 UDP 广播自动发现局域网设备。
   即时共享本地文件——对端通过文件 ID 通过 HTTP 服务下载，无需暴露真实路径。
- ⬇️ **内置下载器**
   集成 **Aria2** 引擎，支持 HTTP、FTP、BitTorrent 等协议。
   提供统一的下载任务列表，支持互联网与局域网任务共管。

## 📦 技术栈

- **前端：** Vue 3 + Vite + TypeScript
- **后端 / 运行时：** Electron + Node.js
- **网络通信：** UDP 广播发现 + HTTP 文件传输
- **下载模块：** Aria2 RPC 集成

## 🚀 跨平台支持

兼容 **Windows**、**macOS**、**Linux** 系统。
 针对离线或内网环境进行了优化。
